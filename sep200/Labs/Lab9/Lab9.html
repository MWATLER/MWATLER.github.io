SEP200 - Lab 9: Car Ratings

Due: April 7, 2023

The Standard Template Library provides a rich set of algorithms that you can perform on your data. For non-standard data types (user defined types) you can provide your own criteria for handling data in the form of a lambda function.

This lab will test your ability to use STL algorithms as you process data from a selection of cars. The information for each car is stored in a structure that contains the car's name, country, reliability, fuel efficiency, and horse power. You might find the following links useful:

  • STL Algorithms
  • STL Numerics
  • STL Containers
  • STL Sort Algorithm
  • STL For-Each Algorithm
  • STL Accumulate Numerics
  • STL Transform Algorithm

The test program CarRating.cpp provides you with a vector of cars with various ratings. The function processCars() asks that you complete the following tasks:

  1. Sort and print out all cars in order of their reliability rating (higher is better).
  2. Sort and print out all cars in order of their fuel efficiency (lower is better).
  3. Sort and print out all cars in order of their horse power (higher is better).
  4. Print out the average horse power of all German cars.
  5. Print out all cars with reliability greater than (or equal to) 8.0.
  6. Change the country of all American cars from "USA" to "United States".
  7. Reorganize the vector of all cars in order of their rating (higher is better) based on the formula:
    rating = 2.0*(reliability-4.1) + 2.0*(12.1-fuelEfficiency) + (horsePower-130.0)/37.0
Complete processCars() according to these instructions.
