  • The files for the Mid-Term are found here.
    The user-level files are memoryUsage.h, fileMon.cpp, memMonitor.cpp, and processMon.cpp.
    The kernel level files are one directory lower, in the directory kernel
    keyGenerator.h, keyGenerator.c, load.sh, and unload.sh.
  • If you want to build the code for yourself, you should start at the kernel level code:
    $ make clean all
    $ sudo load.sh

    You can then build the user level code:
    $ make clean all

    When you are done, be sure to unload this module from the kernel:
    $ sudo unload.sh
