The Corporate Information

A corporation maintains a database of employee information consisting of each employee's name, position, salary and age. The corporation wants to also keep track of some basic statistics such as the average salary, the average age, the maximum salary, and which employee has the maximum salary. Code for populating the corporate information structure has already been presented in the lecture of Week 8. Now it is time to add some database access functions.

The Problem

We want to give the user the ability to perform the following on the database:

  1. Get employee information by employee name
  2. Get employee information for all employees
  3. Correct an employee name given the employee name
  4. Correct an employee age given the employee name
  5. Change an employee position given the employee name
  6. Change an employee salary given the employee name
  7. Commit all changes to the database (write to the file EmployeeDB.txt)

The user should also have the choice to exit the program.

Complete the Code

The complete code for corporation information has been given to you with eight empty functions. You are to complete these functions based on their prototype and embedded instructions. The data structure for the employee can be found in EmployeeInfo.h. The employee database can be found in the file EmployeeDB.txt. The data structure for corporate information and related function prototypes can be found in CorporateInfo.h and the function definitions for the corporation can be found in CorporateInfo.c. The main program is CorporateInfoMain.c.
Complete all eight incomplete functions inside CorporateInfo.c. Some utility functions have been provided to facilitate data input from the keyboard.
