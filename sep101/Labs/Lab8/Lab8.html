Gravity

We are going to simulate the displacement of an object on the moon and an object on the earth. For both places, an object will be dropped from a certain altitude with a constant horizontal velocity. Our program will simulate the x and y displacement of the object as a function of time.

The earth object and moon object are similar, with two differences. The earth object will have a gravitational acceleration constant of 9.81 m/s2 and the moon a gravitational acceleration constant of 1.625 m/s2. The earth object will have no initial vertical speed, but the moon object will.

The earth object will have a gravitational acceleration constant, an initial height, a horizontal speed, an x position and a y position. It can be created through two constructors: one a default constructor and one a constructor which takes as arguments the initial height and the horizontal speed. The earth object will have setter functions to set the initial height and to set the horizontal speed. The setter functions perform validation on the entered values. They will return false if the entered value is less than or equal to 0, true otherwise. The earth object will have getter functions to get the x position (horizontal position) and the y position (vertical position):
x-position = horizontal speed x time
y-position = initial height - 0.5 x gravitational constant x time2

The moon object will have a gravitational acceleration constant, an initial height, an initial vertical speed, a horizontal speed, an x position and a y position. It can be created through two constructors: one a default constructor and one a constructor which takes as arguments the initial height, the initial vertical speed, and the horizontal speed. The moon object will have setter functions to set the initial height, the initial vertical speed, and to set the horizontal speed. The setter functions perform validation on the entered values. They will return false if the entered value is less than or equal to 0, true otherwise. The moon object will have getter functions to get the x position (horizontal position) and the y position (vertical position):
x-position = horizontal speed x time
y-position = initial height + initial vertical speed x time - 0.5 x gravitational constant x time2

Part of the main() function has been provided for you in the file ObjectMain.cpp. You have to complete the TODO section, where you are to prompt the user to enter new parameters for the moon object and the earth object. Be sure to perform validation on the entered values.

For the main() function to work, be sure to put the class declaration for moon inside a file called MoonObject.h and the class declaration for earth inside a file called EarthObject.h. You can put the function definitions for the earth and moon objects inside the files MoonObject.cpp and EarthObject.cpp.
