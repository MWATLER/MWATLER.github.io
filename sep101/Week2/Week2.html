Types

There are many data types in C to accommodate the many types of data required in your programs. C is a typed programming language, that defines how to store values in memory and which operations are admissible on those values. We use different types to store different information.

The four most common types in the C language are: char, int, float, and double.

For more details on data types in C, see Types.

Let us write code to store student information. The data required are the student's last name, the student's identification number, and a list of the student's marks.

  • Code to store and display student information can be found at StudentInformation.c.
  • Be sure to add _CRT_SECURE_NO_WARNINGS to your preprocessor definitions. This warning occurs because of our use of scanf() in our standard input/output library. Visual Studio prefers we use the C++ version. The preprocessor definitions can be found by right-clicking on the project name, then on properties, then on C/C++ Preprocessor. Then click edit on Preprocessor Definitions, then add _CRT_SECURE_NO_WARNINGS.
  • Another way around this problem is to simply add the following line at the top of your code:
    #define _CRT_SECURE_NO_WARNINGS
  • The best way to deal with this is use the safe versions of these functions, such as scanf_s().

Computations

Let us write code to store and display student information, and to calculate the average mark.

Here we are going to keep track of a running total and the number of marks. The running total will be the sum of all the marks. The average is determined by dividing the running total by the number of marks.

For more detail on computations and expressiong in C, see Computations and Expressions.

  • Code to store and display student information including the student's average mark can be found at StudentAverageMark.c.
  • Similar code to retrieve student information from a file and write the student's average mark to a file can be found at StudentAverageMarkFile.c and StudentMarks.txt.

Logic

A computer program has the ability to make a decision based on data that it has received. The selection criteria will determine the different paths through the set of instructions. It is important to follow the principles of structure programming in coding these paths in order to avoid spaghetti code. More detail on selection constructs and structured programming can be found at Logic.

Let us write code to store and display student information, to calculate the average mark, and to convert each mark into a letter grade.

  • Code to store and display student information, to represent each mark as a letter grade, and to also display the average can be found at StudentLetterGrade.c.

Decisions can be concrete. In such cases we may wish to employ a switch() statement instead of an if-else construct.

  • Code to store and display student information, to represent each mark as a letter grade, to display the average and determine the scholarship amount can be found at StudentScholarship.c.
