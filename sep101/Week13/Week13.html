Student List

Let us create a program that reads student data from a CSV file. The file contains the name of a student, the number of subjects, and the mark for each subject. It is not known to the program how many students there are in this file and how many subjects each student is taking.

Each student will have a name, the number of subjects, and an array of marks for each subject. The student will also have an average mark as well as a rank among other students.

The main program will have to open the CSV file and determine how many students are recorded in this file. It will then have to allocate memory for these number of students. It will have to get the name, number of subjects, and the mark for each subject for each student. It will have to calculate the average for each student. After having gone through each student, the main program will print out the information for each student. The program then has to determine the rank for each student. For instance, which student has the highest average, which has the next highest average, etc... Then the program prints the students out again sorted by rank, to the screen and also to a sorted file. The output should be formatted such than the averages are printed to 2 decimal places, the ranks are printed inside a field of 2 characters, the names are printed inside a field of 20 characters, and the averages are printed inside a field of 8 characters.

  • First we identify our class as being of type Student. The student has a name, the number of subjects, the list of marks, the average, and a rank. These should all be represented as variables inside the Student class.
  • There are certain actions required on these variables. We need to calculate the average, get the average, set the student's name, set the marks, get the marks, get the number of marks, get the rank, determine the rank, etc...
  • For a sample implementation, see Student.h, Student.cpp, and StudentList.cpp.
    This implementation reads the CSV file Students.csv and the resulting sorted file is StudentSorted.txt.

C++ Language Standards

  • The C++ language is evolving over time. See C++ Standards for a pictoral summary. A more detailed document can be found at History of C++.
  • There are many libraries that extend the functionality of C++. For example, Blackberry uses the the Adaptive Communication Environment (ACE) library for concurrent communication software.
  • The Boost C++ Libraries offers extensions to every part of the C++ library. See the documentation of their latest release Boost 1.75.0 Library Documentation for more information.
  • For a full list of C++ compatible libraries see cppreference - A list of open source C++ libraries. When you work in a specialized environment, these libraries become indispensible.
