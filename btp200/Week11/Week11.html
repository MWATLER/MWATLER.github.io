Spaceship

  • Source code for a spaceship is studied. Each type of spaceship is based on the base class Spacehip. The types of spaceship are the Cargo vessel, the Starship, and the Warbird. The code demonstrates construction and assignment with derived classes. A main function is given that tests the code in several ways. The UML diagram can be found at:
    Spaceship.PNG.
    The code is:
    Spaceship.h, Spaceship.cpp, Cargo.h, Cargo.cpp, Starship.h, Starship.cpp, Warbird.h, Warbird.cpp, and SpaceTravel.cpp.
