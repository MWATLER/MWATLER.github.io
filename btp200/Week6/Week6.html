Operator Overloading

  • The first program demonstrates the copy constructor and the copy assignment via a circuit simulator which simulates a series electrical circuit with an inductor, resistor and capacitor. Each electrical component is characterized in a class. For instance, the class Inductor records the component's inductance and can calculate the voltage across the inductor for a given current. The code is:
    Inductor.h, Inductor.cpp, Resistor.h, Resistor.cpp, Capacitor.h, Capacitor.cpp, and SeriesCircuit.cpp.
  • The second program shows the overloading of the >> and << operators by a card game played by some of the characters in the TV show "The Sopranos". The code can be found at:
    Player.h, Player.cpp, and CardGame.cpp.

Files

  • A simple program that reads from a file and writes to another can be seen at:
    TextBook.h, TextBook.cpp, and TextBookMain.cpp.
    There are two TextBook objects which read from a file (CheeseShop.txt) that contains the "Cheese Shop Sketch" of the comedy Monty Python. One reads string by string and the other line by line. Both dump their outputs into separate files (CheeseShop2.txt and CheeseShop3.txt).
