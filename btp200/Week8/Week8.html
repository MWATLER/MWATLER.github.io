Sample Code

Calculate Area

  • We will look at the code for quiz 6 in more detail. CalculateArea() calculates the area of a square, rectangle and a circle. The base class is called Shape. The Circle, Rectangle, and Square classes are derived from Shape. The code can be seen at CalculateArea.cpp.

Travel Simulator

  • We looked at a travel simulator earlier in the course, where we drove four cars and measured the fuel left in the tank. This example will use a class Vehicle as the base class and Car and Train as derived classes:
    Vehicle.h, Vehicle.cpp, Car.h, Car.cpp, Train.h, Train.cpp, and TravelSimulator.cpp.
    The code demonstrates function overloading within the base class, calling base class functions from a derived class, and operator overloading outside a class.
