Introduction to Software Engineering

The full software development life-cycle includes requirements, scope, high-level and low-level design, development, testing and rework, deployment, documentation and software maintenance. Software engineering is multi-dimensional in nature. Use-case scenarios, resource limitations, the business context, source code control and quality must be considered.
