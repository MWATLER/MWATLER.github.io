Useful Linux Commands: 18 Popular mount/umount commands

More Useful Linux commands:NOTE: for most of these you have to be super-user
# mount lists all currently mounted devices
# cat /proc/mounts a list of the currently mounted file systems
# cat /etc/mtab slightly more detailed list of currently mounted file systems
# cat /etc/fstab descriptions of all of the available file systems
# mount /dev/sdc1 /u01 mount a device partition
# umount /u01/ unmount a partition
# df -h disk drives on your system
# df -Th disk drives on your system with the types
# mount view all mount information
# mount -V mount command version
# mount -a mount all filesystems of /etc/fstab
# umount -a unmount all filesystems of /etc/fstab
# mount -t tmpfscheck all mounting information of a specific filesystem
# fdisk -l list the entire filesystem
# lsblk -f directory trees for block devices
# parted -lthe partition table
# findmnttarget source for the root directory tree

Sample Code

Mounting /dev/sr0

  • Code that demonstrates mounting and unmounting programmatically mounts /dev/sr0 to a local directory cdrom. The type of the filesystem is RAM. The code can be found at:
    Makefile, Mount.cpp and UMount.cpp.
    Multiple mounting points cdrom0, cdrom1, cdrom2, cdrom3 for dev/sr0 is demonstrated in the following code:
    Makefile, MultipleMount.cpp and MultipleUMount.cpp.
  • Code that demonstrates the binding of mount directories, where /dev/sr0 is mounted to cdrom0 and cdrom1, and cdrom0 and cdrom1 are bound together can be seen at:
    Makefile, BindMount.cpp and MultipleUMount.cpp.

Mounting /dev/sda5 and /dev/sr0

  • Code that mounts /dev/sda5 to cdrom0 with filesystem type ext4, and mounts /dev/sr0 to cdrom1 can be found at:
    Makefile, MountExt4.cpp and MultipleUMount.cpp.
  • Code that gets the statistics for these mounts can be found at:
    Makefile, MountStatvfs.cpp and MultipleUMount.cpp.
