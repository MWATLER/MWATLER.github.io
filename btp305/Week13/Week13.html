MidTerm Review Question

The freighter is a cargo ship that carries a collection of items across the ocean such as cars, food, commodities, etc... Various companies pay for the transportation of their items to markets across the world. For the freighter, a piece of cargo should have a certain value ($10000) to make it profitable to carry. Complete the following program by coding the class named Freighter and a global function named calcTotalWorth.

Freighter Class Requirements

Your Freighter class includes the following members:

  • STL container that holds Cargo objects
  • Total worth of all the cargo
  • Any other utility variable you feel is required
  • Overloaded += operator that
    • Adds a Cargo object to the current object and returns a reference to the current object only if the cargo has a value greater than or equal to $10,000
    • If the value of the cargo is less than $10,000, the operator should not change the current object and throws an exception
  • Overloaded [] operator that
    • Takes as an input the name of a cargo item
    • Returns the address of the cargo object that matches the name
    • If no such cargo exists, the function returns nullptr
  • A display function that:
    • Receives the address of one of the global functions that displays the requested information (byCargoType, byValue, byEverything)
    • Displays the total worth of all cargo
    • Calculates the total worth of all cargo in two threads

calcTotalWorth Function Requirements

  • The global function calcTotalWorth receives iterators to the beginning and end of a range of Cargo objects.
  • This function totals the value of the individual cargo objects

Other Requirements

  • Upgrade the main function of the program to use a smart pointer
  • Upgrade the main function to catch any exceptions
