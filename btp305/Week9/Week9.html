The Stooge Factory

Back in week 3 we looked at the Stooge factory as a demonstration of the factory design pattern. This time we will implement the Stooge factory using the unique_ptr. The Stooge factory creates a set of related objects through a static make_stooge() function. The program is based on the popular comedy "The Three Stooges". The factory can create three stooges: Larry, Curly and Moe. They are tied together through a common base class. StoogeMain.cpp allows you to create whichever stooge you wish, up to three of them:
Stooge.h, Stooge.cpp, Curly.h, Larry.h, Moe.h and StoogeMain.cpp.

Back in week 2 we looked at a Database which was created and used via the Singleton design pattern. This time we will implement the Singleton with a shared_ptr. The Singleton design pattern to ensure there is only one instance of the database class:
Data.txt, Database.h, Database.cpp and DatabaseMain.cpp.

Library of Books (sample code for workshop 7)

A library of books has been created to demonstrate some of the algorithms available to us with the STL. The algorithms demonstrated in this code are std::transform, std::accumulate, std::sort, and std::foreach. The library can sort the books according to the number of pages or according to the price. The library can sum the total number of pages. The library can print its entire contents. The library can convert prices from one currency to another (in this case, USD to CAD). The code should help with workshop 7. It can be seen at:
LibraryBooks.h, LibraryBooks.cpp and LibraryMain.cpp.
