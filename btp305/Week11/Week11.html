Week 11 - The Preprocessor, Multi-Dimensional Arrays

Introduction

The preprocessor performs preliminary operations on C and C++ files before they are passed to the compiler. The preprocessor can conditionally compile code, insert files, specify compile-time error messages, and apply machine-specific rules to sections of code. Multi-dimensional arrays can be tricky to use, especially if they are dynamically allocated. We will look at multi-dimensional arrays with code examples.

The following code creates a 2 dimensional array through dynamic memory allocation. The rows represent students, and the columns represent their marks. To complicate things, each student has a number of marks. So, the array has to be allocated in one direction first: allocate the number of rows, which represents the number of students. Then the number of marks for each student are used to allocate the number of columns for each student. The array is then passed to a function called CurveMarks() to curve each mark for each student.
